### How to Install

#### Clone Repo
`$ git clone <repo_url>`

#### Setup database config (mysql)
- Rename .env_example to .env and change its value

#### Install
`$ npm install`
`$ npm install --save-dev sequelize-cli`
`$ sequelize db:migrate`
